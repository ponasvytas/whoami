TImestamp microservice

Enter date in URL to get unix and natural format

Example
+ https://timestamp-ponasvytas2.herokuapp.com//08 11 1985
+ https://timestamp-ponasvytas2.herokuapp.com/500256000
Example output:
+ {"unix":"500256000","natural":"November 08, 1985"}
Invalid date will show null:
+ { "unix": null, "natural": null }
